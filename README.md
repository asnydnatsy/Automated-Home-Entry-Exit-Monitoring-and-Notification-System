# Automated-Home-Entry-Exit-Monitoring-and-Notification-System
ELZEN (Elder + Zen) is a smart monitoring system designed to enhance the safety of elderly individuals, especially those with cognitive conditions such as dementia. The system detects entry and exit movements, tracks presence status in real time, and notifies caregivers in case of abnormal behavior or prolonged absence.
🧠 How It Works
Two sensors detect movement sequence
System determines entry or exit direction
Updates real-time presence status
Displays status on OLED
Triggers voice feedback via speaker
Monitors duration and behavior patterns
Sends email alert if abnormal condition is detected
